    function sampleDirection

    virtual Interaction sampleDirection(
    const Interaction & ref,
    const Point2f & u,
    float & pdf
) const

    sample a point on the shape and return the pdf with respect to solid angle from given ref

    Parameters:

    • ref reference point
    • u random sample
    • pdf pdf

    \(p(\omega) = \left(\frac{\mathrm d \omega}{\mathrm d x}\right) ^ {-1} p(x)\)

    \(\mathrm d \omega = \mathrm d x \frac{|N(x) \cdot -\vec r|}{||\vec r|| ^ 2}\) where \(\vec r = x - \text{ref}.x\)

    \(\Longrightarrow p(\omega) = \frac{||\vec r|| ^ 2}{|N(x) \cdot -\vec r|}p(x)\)
